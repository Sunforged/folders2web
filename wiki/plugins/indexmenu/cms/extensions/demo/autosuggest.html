BrowserExtensions/DOM/Autosuggest demo
Управляющие кнопки, при курсоре в поле ввода:
Ctrl+space
Arrow Up
Arrow Down
Отображение списка подсказок
PageUp
Pagedown
Home
End
Отображение списка подсказок, если скрыт. При видимом списке - управление перемещением по нему.
Esc
Скрытие списка, если показан. Удаление введённого текста в противоположном случае.
Enter
Выбор подсказки, если список показан. Отправка формы на сервер в противоположном случае.
Ввод 2 и более латинских букв
Отображение списка подсказок, если скрыт. Фильтрация подсказок по введённым буквам.
Backspace
Отображение списка подсказок, если в поле после удаления остаётся более 2 символов и в списке подсказок есть соответствия.
 Control keys:
Ctrl+space
Arrow Up
Arrow Down
Display suggestions list
PageUp
Pagedown
Home
End
Display suggestions list, when hidden. Otherwise - contol the list movement.
Esc
Hide suggestions list, when visible. Delete entered text otherwise.
Enter
Paste suggestion in the field, when the list is visible. Submit form otherwise.
Typing 2+ letters
Display suggestions list, when hidden. Filter suggestions.
Backspace
Display suggestions list, when after deletion 2+ letters are still available and list has the matches.
